📝 Módulos Operacionales

Captura automática de datos: OPC → Hosteo → Worksheet → Contrato

Flujo de Datos: Cada módulo pre-rellena el siguiente automáticamente. Los datos se almacenan en localStorage y se transfieren entre formularios.
🎁

Módulo OPC

Invitaciones: quién invita, a quién, cuándo, dónde, regalos ofrecidos

🏨

Módulo Hosteo

Datos amplios: domicilio, contacto, trabajo, ingresos, tarjetas (últimos 4), IDs, viajes

📋

Módulo Worksheet

Membresía + Calculadora VCM (20% descuento) + Firma digital + Genera contrato PDF automático

⚙️ Panel de Configuración

Edita todos los parámetros sin tocar código: precios, descuentos, comisiones, metas

Super Funcional: Sliders para %, inputs numéricos, vista previa en tiempo real, exporta/importa configuración como JSON
💰

Gestión de Precios

Edita precios base (Premium/Platinum/Ultra/Vitalicia), calcula automáticamente duraciones

📉

Descuentos VCM

Slider para % de descuento (20% default) con vista previa en vivo del ahorro

💵

Comisiones

Estructura: OPC + Hosteo + Contrato + Pago con ejemplos en vivo de ventas

🎯

Metas Mensuales
