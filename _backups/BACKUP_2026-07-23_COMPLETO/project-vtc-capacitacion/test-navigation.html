Próximos Pasos

  1. Copiar los 3 archivos (index-scroll-navigation.html, styles-scroll-nav.css, main-scroll-nav.js) a tu carpeta de deploy
  2. Crear carpeta `/videos/` con los archivos de video y posterimages (modulo-01.jpg, modulo-01.mp4, etc.)
  3. Abre el archivo HTML en navegador moderno (Chrome 90+, Safari 14+, Firefox 88+, Edge 90+)
  4. Prueba scroll horizontal: swipe derecha/izquierda en desktop o mobile
  5. Verifica responsive: abre DevTools y cambia viewport size
  6. Configura .env con ELEVENLABS_API_KEY si necesitas integración de agent
  7. Monta endpoint POST /api/elevenlabs-agent en tu backend
  8. Deploy a producción (Vercel, Netlify, self-hosted, etc.)

Quick Start (Sin Backend)

Para probar el frontend sin backend ElevenLabs:

$ npx http-server /path/to/deploy Server running at http://127.0.0.1:8080 Open http://127.0.0.1:8080/index-scroll-navigation.html in browser

Las llamadas a ElevenLabs se ignoran gracefully si no hay backend configurado.
