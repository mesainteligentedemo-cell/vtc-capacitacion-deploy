## 🎯 PRONUNCIACIÓN POR MÓDULO

### Módulo 0: OPC (Offsite Personal Contact)
**Palabras Críticas:**
- HOSTEO → JOSTEO
- OPC → O-PE-CE
- TIMESHARE → TAIM-SHER
- INVITACIÓN → in-vi-ta-SIÓN

### Módulo 1: WORKSHEET (Contrato)
**Palabras Críticas:**
- WORKSHEET → WORK-SHEET
- MEMBRESÍA → mem-bre-SÍA
- PLATINUM → PLA-TI-num
- FINANCIAMIENTO → fi-nan-sia-MI-EN-to

### Módulo 2: HOSTEO (Datos Ampliados)
**Palabras Críticas:**
- HOSTEO → **JOSTEO** (OBLIGATORIO)
- DOMICILIO → do-mi-CI-lio

## 🔧 CÓMO USAR EN ELEVENLABS

**Opción 1: Usar Pronunciación Manual**
1. En ElevenLabs, ve a Settings
2. Busca "Pronunciation Dictionary"
3. Agrega cada palabra de arriba
4. Copia la "Pronunciación IPA"
5. Guarda y prueba

**Opción 2: Usar XML en el Prompt**
```xml
<phoneme alphabet="ipa" ph="xos'teo">HOSTEO</phoneme>
Se pronuncia <phoneme alphabet="ipa" ph="xos'teo">HOSTEO</phoneme>
```

**Opción 3: Editar Directamente en Script**
Reemplaza en el guion:
```
❌ HOSTEO
✅ JOSTEO
```
